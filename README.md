# CodeAlpha_Simple ECommerce Store
A modern and responsive *E-Commerce Store* developed using the *MERN Stack* during the *CodeAlpha Full Stack Development Internship*.  
This project provides a smooth online shopping experience where users can browse products, add items to cart, and manage shopping activities through an interactive interface.

---

## 🚀 Features

✅ Product Listing Page  
✅ Add to Cart Functionality  
✅ Remove Items from Cart  
✅ Product Details Section  
✅ Responsive User Interface  
✅ Dynamic Product Management  
✅ Backend API Integration  
✅ Clean & Modern Design
